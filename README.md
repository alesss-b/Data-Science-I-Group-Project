# Data Science I Group Project by Fariha Adil, Alex Bradshaw, and Julien Garcia

This project explores the DBLP computer science bibliography dataset through three analyses: citation network analysis with author influence, co-authorship network and research community detection, and identifying emerging research topics. Each analysis is contained in its own Jupyter Notebook.

## Requirements

We created this project in Python in Jupyter Notebooks, so please make sure you have both Python and Jupyter Notebook installed before running the code! If Python is not installed, you can download it from https://www.python.org/downloads/. You can install Jupyter Notebook through python -m pip install notebook.

We used numpy, pandas, matplotlib, seaborn, networkx, scikit-learn, scipy, and python-louvain in this code. These are all packages that need to be installed on your computer for this program to work. You can install them using pip in the command line. To do this, open PowerShell or another terminal and type `python -m pip install package-name`. Alternatively, you can install all 
dependencies at once by running `pip install -r requirements.txt` in the project 
root (requirements.txt is included in the zip).

## Dataset

Before running any of the notebooks, you must download and extract the [dataset](https://canvas.uh.edu/courses/29263/files/9886365?wrap=1) into a folder called `dblp_ref/` in the project root. Make sure the four JSON files (dblp-ref-0.json through dblp-ref-3.json) are inside that folder and named correctly, as this will cause the code to fail otherwise.

## Running the Code

Run the notebooks in this order:

1. `data_preprocessing.ipynb` — loads and cleans the full dataset and saves `cleaned_dataset.pkl` for use by the citation network and co-authorship notebooks
2. `citation_network_analysis.ipynb` — builds the citation network and runs influence analysis, saving results to `outputs/` and `figures/`
3. `co-authorship_network_and_research_community_detection.ipynb` — builds the co-authorship network and detects research communities using Louvain clustering
4. `identifying_emerging_research_topics.ipynb` — loads directly from the raw JSON files, runs TF-IDF and NMF topic modeling, and saves results to `outputs/`

Notebooks 2 and 3 require `cleaned_dataset.pkl` to be generated by notebook 1 first. Notebook 4 loads the raw data independently and can be run at any time after the dataset is in place.

## How It's Made

This project was built in Python with the numpy, pandas, matplotlib, seaborn, networkx, scikit-learn, scipy, and python-louvain packages. We used pandas to load and preprocess the dataset. We used networkx to build and analyze the citation and co-authorship graphs, and python-louvain to detect research communities using the Louvain algorithm. We used scikit-learn to build TF-IDF features and apply NMF topic modeling. We used matplotlib and seaborn to create all visualizations.

## Lessons Learned

We learned how graph-based metrics like PageRank can reveal influence patterns in large citation networks that raw citation counts alone cannot capture. We also learned how co-authorship networks can be used to identify tight-knit research communities and how topic modeling can uncover latent themes in academic literature and track how they evolve over time.
